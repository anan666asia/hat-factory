## 🎮 在线体验

👉 **[点击开始测试](https://anan666asia.github.io/hat-factory/)**

测测你被扣过什么帽！12道情境测试题，匹配「黑手」「野心家」「反党分子」等经典网络梗帽子。

## ✨ 功能特性

- 🎯 **12道情境测试** - 逐题展示，答完自动下一题
- 🎨 **炫酷结果页** - 全屏展示MBTI类型 + 专属帽子
- 📋 **16型罪状名录** - 查看所有类型的帽子解析
- 📱 **完美适配手机** - 响应式设计，随时随地测
- ⌨️ **键盘快捷键** - 方向键切换，1/2选答案
- 📋 **一键复制结果** - 分享给朋友

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/anan666asia/hat-factory.git

# 进入目录
cd hat-factory

# 用浏览器打开
open index.html
```

或者直接用 Live Server 等工具预览。

## 📝 测试流程

1. 点击「开始测试」进入欢迎页
2. 逐题选择最符合你的选项（共12题）
3. 答完最后一题自动显示结果
4. 查看你的MBTI类型 + 专属帽子
5. 复制结果或查看完整罪状名录
